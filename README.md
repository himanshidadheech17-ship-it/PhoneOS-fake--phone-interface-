 PhoneOS — Fake Phone Interface (Pure HTML & CSS)

A static smartphone UI mockup built with **only HTML and CSS** — no JavaScript, no frameworks. Every home-screen icon opens a real page styled to look like a working app.

**Live Demo:** _add your deployed link here after deploying_

##  Apps

| App | What it shows |
|---|---|
|  Camera | Viewfinder mockup with shutter button |
|  Gallery | Grid of placeholder photos |
|  Notes | Sample note cards |
|  Messages | Static conversation list |
|  Calculator | Full calculator keypad UI |
|  Settings | Pure-CSS toggle switches, profile, about |
|  Phone | Recent calls + dial pad |
|  Browser | Address bar + bookmarks grid |
|  Music | Now-playing screen with progress bar |

> As this is build with HTML/CSS only, buttons and toggles are visual — they don't perform real actions (no calculations, no saved notes, no live camera). The Settings toggle switches are  clickable, styled with a pure CSS checkbox trick.

##  Built With

- HTML5
- CSS3 (custom properties, Grid, Flexbox, `:checked` selector for toggles)

## Running Locally

Open `index.html` in any browser — no server, no build step needed. For the smoothest experience in VS Code, use the Live Server extension.

## Deployment

Static site — deploy free on Netlify (drag-and-drop the folder) or GitHub Pages (Settings → Pages → source: `main` branch).
